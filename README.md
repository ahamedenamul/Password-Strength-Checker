# Password-Strength-Checker
Password Strength Checker with HTML, CSS, and JavaScript - Practice Project 
"A real-time Password Strength Checker application built with HTML, CSS, and JavaScript that validates password complexity visually."

Password Strength Checker 

This is an interactive web application that checks the strength of a user's password. It provides real-time feedback based on password length, letters, numbers, and special characters.

## Features
- Real-time checking: Checks the strength of the password as you type.
- Visual indicator: Colorful progress bar to show how strong the password is (Weak, Medium, Strong).
- Password toggle: Show/Hide the password.
- Conditions check: Checks for lowercase letters, uppercase letters, numbers, and symbols.
- Responsive design: Beautiful and modern user interface.

## Technologies Used
- HTML5
- CSS3 (Flexbox, Transitions, Gradient)
- JavaScript (DOM Manipulation, Regex)

## How to use
1. Start typing the password in the input box.
2. If the requirements below are met, a green check mark will appear.
3. If the password is strong, the bar will fill up and turn green.
